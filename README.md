# Malaanta Website

**Official website for Malaanta** — A smart waste collection and management platform based in Kathmandu, Nepal.

## About Malaanta

Malaanta is a waste technology company building a smarter, cleaner, and more sustainable waste management system for Nepal.

> “फोहोर र मैलो, सुनको थैलो”  
> *(Waste and dirt, a bag of gold)*

We focus on **source segregation**, efficient collection, and turning waste into valuable resources through composting and recycling — helping Nepal move towards a circular economy.

## Live Website

🌐 [https://malaanta-website.vercel.app](https://malaanta-website.vercel.app)

## Features

- Modern, clean, and responsive design
- Bilingual-friendly (Nepali + English)
- Services for Households, Businesses & Institutions
- Information on source segregation (Red, Blue, Green bags)
- Vision for 2030 (Zero landfill, green jobs, digital tracking)
- Contact and service information
